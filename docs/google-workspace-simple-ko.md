# VOXERA 구글 워크스페이스 쉬운 버전

## 결론

맞습니다. 이전 버전은 초보 사용자용이 아니라 운영자/개발자용에 가까웠습니다.
한국 초보 사용자를 위한 기본 템플릿은 더 단순해야 합니다.

이 쉬운 버전은 아래처럼 바꿉니다.

### 사용자에게 보이는 탭
- `대시보드`
- `받은 음성함`
- `실행 보드`
- `설정`

### 숨겨두는 탭
- `보관함`
- `대시보드 데이터`

즉, 사용자는 처음에 **딱 4개 탭만** 보면 됩니다.

## 왜 이렇게 바꾸는가

Google 공식 도움말 기준으로 초보자에게 친화적인 시트는 아래 기능이 핵심입니다.

- 드롭다운 칩: 상태/우선순위를 클릭으로 선택 가능 [공식 도움말](https://support.google.com/docs/answer/12319513?hl=en)
- 날짜 입력: 셀에 날짜를 쉽게 넣을 수 있음 [공식 도움말](https://support.google.com/docs/answer/12319513?hl=en)
- 조건부 서식: 중요한 항목을 눈에 띄게 표시 가능 [공식 도움말](https://support.google.com/docs/answer/78413?hl=en)

즉, 한국 초보 사용자용 기본형은
- 컬럼 수 적게
- 탭 수 적게
- 전부 한국어
- 클릭형 드롭다운
이 맞습니다.

## 쉬운 버전 구조

### 1. 받은 음성함
여기는 검토용입니다.

컬럼:
- 받은 시각
- 요청 ID
- 말한 사람
- 한 줄 요약
- 실행 항목
- 우선순위
- 마감일
- 문서 필요
- 일정 필요
- 상태
- 처리 시각
- 메모

상태:
- `검토전`
- `실행전환`
- `보관`
- `오류`

### 2. 실행 보드
여기가 실제 실행용입니다.

컬럼:
- 할 일
- 담당자
- 우선순위
- 상태
- 마감일
- 문서
- 일정
- 완료일
- 요청 ID
- 요약

설명:
- `문서` 셀은 긴 URL이 아니라 `문서 열기` 버튼처럼 보입니다
- `일정` 셀은 긴 URL이 아니라 `일정 보기` 버튼처럼 보입니다
- 클릭하면 바로 문서/일정이 열립니다

상태:
- `해야함`
- `진행중`
- `대기중`
- `완료`

우선순위:
- `높음`
- `보통`
- `낮음`

### 3. 설정
여기만 채우면 됩니다.

항목:
- 기본 담당자
- 문서 폴더 ID
- 캘린더 ID
- 알림 메일 사용
- 알림 메일 주소
- 웹훅 비밀키
- 기본 일정 시작 시각
- 기본 일정 길이(분)

### 3. 대시보드
여기는 보기용 첫 화면입니다.

보이는 것:
- 검토할 음성 수
- 진행 중 업무 수
- 오늘 마감 수
- 완료한 업무 수
- 오늘 바로 봐야 할 실행 항목

즉, 사용자는 시트를 열자마자 표가 아니라 **작은 대시보드 카드**를 먼저 보게 됩니다.

## 사용 방법도 더 단순하다

1. 외부에서 데이터가 `받은 음성함`으로 들어온다
2. 사용자가 `한 줄 요약`과 `실행 항목`을 본다
3. 진짜 해야 할 일이라고 판단되면 `상태`를 `실행전환`으로 바꾼다
4. 그러면 `실행 보드`에 자동으로 들어간다
5. 문서가 필요하면 `문서 열기` 버튼 생성
6. 일정이 필요하면 `일정 보기` 버튼 생성

## 이 버전이 더 맞는 이유

- 처음 보는 사람도 탭 3개만 이해하면 된다
- 모든 컬럼이 한국어다
- 메타데이터를 앞에 드러내지 않는다
- 실행 보드는 정말 필요한 컬럼만 남겼다
- 문서/일정은 필요한 경우에만 붙는다

## 왜 더 대시보드처럼 보이게 했나

Google Sheets는 완전한 앱 UI는 아니지만, 아래 공식 기능들로 충분히 더 친화적으로 만들 수 있습니다.

- 드롭다운 칩 [Google 공식 도움말](https://support.google.com/docs/answer/12319513?hl=en)
- 조건부 서식 [Google 공식 도움말](https://support.google.com/docs/answer/78413?hl=en)
- 링크 텍스트/리치 텍스트 [Apps Script RichTextValue](https://developers.google.com/apps-script/reference/spreadsheet/rich-text-value-builder)

그래서 이번 버전은:
- 상태 색상 구분
- 클릭 가능한 짧은 칩
- 첫 화면 대시보드 카드
- 숨김 시스템 탭

구조로 바꿨습니다.
