# B2B Cold Landing Bento - Visual PRD

## Brief Restatement
Create a high-end enterprise landing blueprint in the Linear/Vercel family: deep navy atmosphere, mesh-like technical depth, decisive hero messaging, trust marquee, and a 6-column bento proof grid centered on quantified business outcomes.

## S.T.R.U.C.T

### S. Screen
- Theme: deep navy enterprise tech surface
- Bias: desktop-first B2B presentation that still compresses to mobile
- Ratio: wide hero and broad grid page with clean top navigation
- Viewport mood: precise, premium, performance-oriented

### T. Tree
- Root shell
  - mesh gradient background layer
  - top navigation bar
  - hero section
    - neon proof badge
    - headline block
    - supporting copy
    - CTA cluster
    - stat rail
  - marquee trust bar
  - 6-column bento grid
    - large chart card
    - automation flow card
    - governance/security card
    - integrations card
    - team velocity card
    - ROI summary card
  - closing CTA strip

### R. Relative
- Top nav spans full width while content sits inside a centered `max-w-6xl` frame
- Hero content aligns left-of-center with supporting stat rail to the right or beneath on smaller screens
- Trust marquee acts as a full-width separator band between hero and bento grid
- Bento grid uses one dominant oversized card and five supporting cards arranged around it
- Closing CTA strip anchors the page bottom with centered action emphasis

### U. Units
- Container: `max-w-6xl` primary shell, `max-w-md` collapse targets on mobile cards
- Spacing rhythm: balanced enterprise spacing with deliberate grid gutters
- Grid density: medium-tight inside cards, spacious between major sections
- Card padding: 24-32px desktop, 18-20px mobile
- Text wrapping rule: hero and metric copy should use `break-keep`

### C. Components
- Nav: brand, product links, CTA button
- Neon badge: provocative positioning statement
- Hero headline: anti-assistant, pro-automation enterprise claim
- Support copy: operational outcome explanation
- CTA cluster: book demo and see ROI
- Stat rail: compact performance highlights
- Marquee: trust/customer signal strip
- Bento cards: KPI chart, workflow automation, governance, integrations, team efficiency, ROI
- Chart mockup: upward-right line chart in the largest card
- Closing CTA strip: final conversion block

### T. Theme
- Tone: cold, exact, confident
- Surface: translucent navy cards with 1px white alpha borders and radial inner glows
- Hierarchy: high in hero, medium-high in bento metrics
- Motion mood: subtle marquee drift and low-amplitude mesh ambience
- Contrast: bright white text, electric cyan/violet accents, restrained gradients

## SVG Blueprint Intent
- Make the nav, hero, marquee, and 6-column bento system immediately readable
- Emphasize the dominant KPI card with the 92% email-time reduction chart mockup
- Preserve enterprise clarity over decorative noise
